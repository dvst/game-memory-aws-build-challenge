AWS Memory Game - Juego Educativo de Servicios Amazon Web Services

Aprende AWS Jugando - Modo Fácil y Difícil

Descubre los servicios más importantes de Amazon Web Services a través de nuestro innovador juego de memoria con estilo retro 8-bits. Perfecto para estudiantes, desarrolladores y profesionales de la nube.

Servicios AWS Incluidos:

  • Amazon S3 - Simple Storage Service para almacenamiento en la nube
  • AWS Lambda - Funciones serverless para computación sin servidor
  • Amazon EC2 - Elastic Compute Cloud para servidores virtuales
  • Amazon RDS - Relational Database Service para bases de datos
  • Amazon EKS - Elastic Kubernetes Service para contenedores
  • Amazon SageMaker - Plataforma de machine learning
  • Amazon Kinesis - Streaming de datos en tiempo real
  • Amazon Redshift - Data warehouse para análisis
  • AWS CloudFormation - Infraestructura como código
  • Amazon ElastiCache - Cache en memoria distribuido
  • Amazon SNS - Simple Notification Service
  • Amazon SQS - Simple Queue Service
  • Amazon DynamoDB - Base de datos NoSQL
  • Amazon CloudWatch - Monitoreo y observabilidad

Características del Juego:

  • Dos niveles de dificultad: Fácil (8 cartas) y Difícil (18 cartas)
  • Interfaz retro estilo 8-bits inspirada en Nintendo
  • Música de fondo generada programáticamente
  • Soporte multiidioma: Español e Inglés
  • Completamente gratuito y sin registro
  • Compatible con dispositivos móviles y desktop
  • Efectos de sonido auténticos de arcade
  • Colores oficiales de AWS para cada servicio

Beneficios Educativos:

Este juego ayuda a memorizar y familiarizarse con los servicios principales de AWS de manera divertida y efectiva. Ideal para:

  • Preparación para certificaciones AWS
  • Estudiantes de computación en la nube
  • Desarrolladores que inician en AWS
  • Profesionales de DevOps y arquitectura cloud
  • Cualquier persona interesada en aprender sobre AWS

Tecnologías Utilizadas:

  • HTML5 Canvas para renderizado de gráficos
  • JavaScript ES6+ para lógica del juego
  • Phaser.js 3.70.0 como motor de juegos 2D
  • Web Audio API para efectos de sonido
  • CSS3 con animaciones y efectos visuales
  • API de GitHub para datos en tiempo real
